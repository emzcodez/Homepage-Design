# Homepage-Design
This project implements a responsive homepage for the Institute of Digital Risk (IDR) along with a minimalist brand logo. The logo uses a geometric cube design to represent structure, resilience, and interconnected digital systems. The orange color highlights innovation and forward-thinking in digital and AI risk management, while black and white convey professionalism and trust.

The webpage was built using semantic HTML5 and CSS, following a clean and minimalist design style aligned with the required orange, black, and white color palette. The layout uses CSS Grid for the services section and Flexbox for navigation and button alignment, allowing content to be structured clearly and consistently.

Several usability features were implemented, including a sticky navigation bar with smooth scrolling, call-to-action buttons, and hover effects to improve user interaction. The site is also fully responsive, using media queries to adjust layouts for mobile devices, such as stacking service cards and navigation links vertically.

Overall, the design focuses on clarity, accessibility, and responsiveness while demonstrating fundamental front-end development practices using vanilla HTML, CSS, and responsive design techniques.
